## ✅ 已完成的准备工作

1. ✅ Git 仓库已初始化

## 📝 接下来的步骤

### 1. 在 GitHub 创建仓库

访问：https://github.com/new

填写信息：
- **Repository name**: `zen-particles-3d`
- **Description**: `❤️ 3D爱心粒子手势控制 - 使用MediaPipe和Three.js`
- **Public** (公开仓库才能使用免费的GitHub Pages)
- ⚠️ **不要**勾选 "Add a README file"
- ⚠️ **不要**勾选 "Add .gitignore"

点击 **Create repository**

### 2. 推送代码到 GitHub

复制以下命令并执行（替换 YOUR_USERNAME 为你的GitHub用户名）：

```bash
# 添加远程仓库
git remote add origin https://github.com/YOUR_USERNAME/zen-particles-3d.git

# 重命名分支为 main
git branch -M main

# 推送代码
git push -u origin main
```

### 3. 启用 GitHub Pages

1. 进入你的 GitHub 仓库页面
2. 点击 **Settings** (设置)
3. 左侧菜单找到 **Pages**
4. **Source** 部分选择 **GitHub Actions**
5. 保存

### 4. 等待自动部署

- 推送代码后会自动触发部署
- 查看部署状态：仓库页面 → **Actions** 标签
- 等待绿色 ✅ 表示部署成功（约2-3分钟）

### 5. 访问你的网站

部署成功后，访问：
```
https://YOUR_USERNAME.github.io/zen-particles-3d/
```

## 📞 遇到问题？

### GitHub 推送失败
- 检查网络连接
- 确认 GitHub 用户名和仓库名正确
- 可能需要配置 SSH 密钥或使用 Personal Access Token

### 部署失败
- 查看 GitHub Actions 日志
- 确认 `package.json` 依赖完整
- 检查 Node 版本（需要 20+）

### 网站无法访问
- 等待 2-3 分钟让 DNS 生效
- 清除浏览器缓存
- 检查 GitHub Pages 设置是否正确
